Hint 3: Full Fix

Delete and recreate the pod with the corrected volumeMount:

  kubectl delete pod config-app -n ckaquest

  cat <<EOF | kubectl apply -f -
  apiVersion: v1
  kind: Pod
  metadata:
    name: config-app
    namespace: ckaquest
  spec:
    containers:
      - name: config-app
        image: busybox:1.36
        command: ["sh", "-c", "cat /etc/app/app.conf && sleep 3600"]
        volumeMounts:
          - name: config-vol
            mountPath: /etc/app/app.conf
            subPath: app.conf
    volumes:
      - name: config-vol
        configMap:
          name: app-config
  EOF

The key change: mountPath is now the full file path and subPath selects
the single key from the ConfigMap.

The node label was set to zone=us-west by the mission setup.
The pod must tolerate that zone.

Quick fix:
  kubectl delete pod zone-aware-app -n ckaquest
  kubectl apply -f - <<EOF
  apiVersion: v1
  kind: Pod
  metadata:
    name: zone-aware-app
    namespace: ckaquest
    labels:
      app: zone-aware-app
  spec:
    affinity:
      nodeAffinity:
        requiredDuringSchedulingIgnoredDuringExecution:
          nodeSelectorTerms:
            - matchExpressions:
                - key: zone
                  operator: In
                  values:
                    - us-west
    containers:
      - name: app
        image: busybox:1.36
        command: ["sh", "-c", "echo 'Running in correct zone'; sleep 3600"]
  EOF

Verify:
  kubectl get pod zone-aware-app -n ckaquest
  # Running

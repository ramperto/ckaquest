Complete rollback:

  kubectl rollout undo deployment/api -n ckaquest
  kubectl rollout status deployment/api -n ckaquest

Verify the image is restored:
  kubectl get deployment api -n ckaquest \
    -o jsonpath='{.spec.template.spec.containers[0].image}'
  # Should print: nginx:1.25

Check all pods are healthy:
  kubectl get pods -n ckaquest -l app=api
